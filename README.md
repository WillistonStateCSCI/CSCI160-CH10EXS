# CSCI160-CH10EXS
Programming Examples From Java Illuminated Book by Anderson and Franceschi
## Example Descriptions:  
**Example 10.01, 10.02 and 10.03** - A checking account client that uses BankAccount.java and CheckingAccount.java  
**Example 10.04, 10.05 and 10.06** - An update to the prior example that uses overloaded constructors  
**Example 10.07 and 10.08** - An updated client that charges a monthly fee  
**Example 10.09 and 10.10** - An updated client with changes to the toString operator  
**Example 10.11, 10.12 and 10.13** - An updated client with access changes  
**Example 10.21, 10.22, 10.23, 10.24 and 10.25** - A hex string handler  
**Example 10.26, 10.27, 10.28, 10.29 and 10.30** - A shipping calculator
